# Custom Video Player — Brad Traversy 50 Projects

Custom HTML5 video player with JavaScript controls.  
This project replicates the exercise from Brad Traversy’s *50 Projects in 50 Days* course.

## Features
- Play and pause video  
- Update progress bar  
- Set video progress manually  
- Adjust volume  
- Toggle fullscreen mode  

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  
- HTML5 `<video>` element  

## Project Purpose
Practice working with media elements, custom controls, and event listeners.

## How to Use
1. Open `index.html` in your browser.  
2. Use the custom controls to interact with the video.
